# Tag Taxonomy (v0.1)

Tags come from three controlled axes. A node typically has 2–5 tags,
at least one from Paradigm or Topic Area.

## 1. Paradigm (theoretical lens)
structural-functionalism, conflict-theory, symbolic-interactionism,
feminist-theory, rational-choice, postmodernism

## 2. Level of analysis
macro, meso, micro

## 3. Topic area (aligned to OpenStax chapters)
foundations, culture, socialization, groups-organizations,
deviance-crime, media-technology, inequality, race-ethnicity,
gender-sexuality, aging, family, religion, education,
health-medicine, government-economy, population-environment,
social-movements, research-methods

## Adding a new tag
Propose via GitHub Issue with: the tag, its axis, definition,
and 2+ nodes that would use it. Tags are kebab-case, singular
where possible.
