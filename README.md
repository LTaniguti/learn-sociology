# learn-sociology
An open-source, graph-navigable platform for self-learning Sociology.
# about
This is intented to be an open source platform for users to be able to self-learn Sociology. The final version of the platform is intended to provide four navigational modes for course content: 
1. An ordered structure, reminiscent of introductory university courses.
2. A concept map formatted simliarly to a hierarchy chart, where more core concepts eventually branch out into more niche concepts.
3. A navigatable network consisting of concepts and theories that visualizes how they are interconnected and relate to each other.
4. A navigatable network of sociologists that visualizes the scale of contributions and how the work they have done is interconnected.

# status
This project is currently in early scoping, and is not yet usable. Contributions will be very much welcome once the framework is further defined.
